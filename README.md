## Descripción del proyecto
Es un Cajero Bancario al que debemos introducir nuestro nombre y nuestra contraseña para acceder a la pantalla de menú.
El nombre puede ser el de nuestra preferencia (ejemplo Pepe, Juan, Fulgencio, Pancracio, etc.).
La contraseña es 1234.
El menú tiene comandos para consultar saldo, ingresar efectivo y pagar deudas domiciliadas.
Las deudas domiciliadas son LUZ y AGUA.
## Situación actual del proyecto
El algoritmo está incompleto para el pago del AGUA. Sólo paga la deuda domiciliada en la LUZ.
## Misión del estudiante
Debe proponer una mejora al algoritmo para completarlo y se pueda pagar también la deuda del AGUA.
Realizada la mejora, el estudiante pedirá un PUSH REQUEST para que la plataforma GitHub habilite el respectivo COMMIT.
